Agrupamento de Seletores

Podemos aplicar as mesmas regras CSS para diversos seletores diferentes. Para isso podemos separar os seletores por uma vírgula(,)

Assim, evitamos o Ctrl+C e Ctrl+V para os códigos, podemos fazer um só bloco de CSS e reutilizar as mesmas regras
